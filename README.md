# web_hangman_flask

## Run Hangman App:

1. Clone the repo.
2. In the command line:

* python3 -m venv env
* source env/bin/actuvate
* pip install -r requirements.txt
* flask run

 That's all!

 TIP: make sure you have installed 'pip'
